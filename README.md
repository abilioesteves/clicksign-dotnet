# Clicksign for .Net

Biblioteca em .Net para consumo do serviços da Clicksign.

A c é uma solução online para enviar, guardar e assinar documentos, com validade jurídica. Foi criada para facilitar, reduzir custo e aumentar a segurança e compliance do processo de assinatura e workflow de documentos. Documentação da API pode está disponível em <a href="https://github.com/clicksign/rest-api-v2" target="_blank">https://github.com/clicksign/rest-api-v2</a>

Atuamente a biblioteca <a href="https://github.com/adrianocaldeira/clicksign-for-dotnet">Clicksign for .Net</a> está em acordo com a versão mais recente da Clicksign.
